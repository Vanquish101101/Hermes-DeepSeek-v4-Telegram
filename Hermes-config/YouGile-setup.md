# YouGile MCP — настройка и архитектура учёта задач (добавлено 2026-06-22, версия 1.8.0)

YouGile подключён к Hermes как MCP (пакет `@nebelov/yougile-mcp`, 57 инструментов, полное покрытие
API v2) с 16.06.2026, но до этой версии не имел роли в оркестрации.

## Где настроено (файл в `AppData\Local\hermes\`, в репо не коммитится — содержит ключ)
**`config.yaml` → `mcp_servers.yougile`**:
```yaml
yougile:
  command: npx
  args: ["-y", "@nebelov/yougile-mcp"]
  enabled: true
  env:
    YOUGILE_API_KEY: Z7BeP9…   # ключ скрыт; реальный — в config.yaml
```
Ключ получен через REST API (в интерфейсе YouGile нет кнопки генерации):
1. `POST https://ru.yougile.com/api-v2/auth/companies {login, password}` → `companyId`
2. `POST https://ru.yougile.com/api-v2/auth/keys {login, password, companyId}` → API-ключ
